# 🔎 Inverted Search using Data Structures in C

## 📌 Project Overview

The **Inverted Search** project is a text search system developed using **C programming and Data Structures**.

It creates an **inverted index** from multiple text files and allows users to **search words efficiently** across documents.

This project demonstrates **hashing**, **linked lists**, and **file handling** concepts.

---

## 🚀 Features

✨ Create inverted index database  
✨ Search words across multiple files  
✨ Display stored database  
✨ Update database dynamically  
✨ Save database to file  
✨ Efficient word searching  
✨ Supports multiple input text files  

---

## 🛠️ Technologies Used

- Programming Language: **C**
- Data Structures:
  - Hash Tables
  - Linked Lists
- Concepts:
  - File Handling

  - Searching Algorithms
